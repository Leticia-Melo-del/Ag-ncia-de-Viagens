# Horizonte Livre Turismo

Projeto desenvolvido em HTML e CSS para a atividade de Desenvolvimento Web.

## Sobre

Esta página institucional apresenta a empresa fictícia **Horizonte Livre Turismo**, com foco em um visual organizado, moderno e de fácil manutenção usando **CSS externo**.

## Estrutura do projeto

```text
AtividadeCss/
|-- index.html
|-- style.css
`-- README.md
```

## Tecnologias utilizadas

- HTML5
- CSS3

## Como visualizar

1. Abra a pasta do projeto.
2. Dê dois cliques no arquivo `index.html`.
3. O navegador irá carregar a página com os estilos definidos em `style.css`.

## Estrutura da página

- `header`: título e apresentação da empresa
- `nav`: menu de navegação
- `section.container`: área principal do conteúdo
- `.card`: blocos de conteúdo
- `footer`: informações de contato

## Requisitos atendidos

- Uso de arquivo CSS externo
- Tema adaptado para **Agência de viagens**
- Menu com pelo menos 3 itens
- Uso de `header`, `nav`, `section`, `.container`, `.card` e `footer`
- Uso de **Flexbox**
- Uso de efeito `:hover`
- Personalização de cores, textos e estilo visual
